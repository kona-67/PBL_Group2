# PBL_Group2

・controllerディレクトリ
GET・POSTなどの処理を書いたJavaファイル
を入れるディレクトリ


・entityディレクトリ
DBのテーブルを作るJavaファイル
を入れるディレクトリ


・formディレクトリ
変数を定義する・変数に格納する値の基準(ルール)を定義するJavaファイル
を入れるディレクトリ


・repositoryディレクトリ



・templateディレクトリ
この中にHTMLファイルを入れる予定です。


・staticディレクトリ
この中にJavaScriptとCSSを入れる。

・targetディレクトリ
コンパイルファイルをディレクトリ別に格納している。
コンパイルファイル：Javaファイルをコンピューターが理解できる文章に書き直したファイル。
コンピューターが使うファイルが入っているので、僕らは使いません。


・pom.xmlファイル
Mavenという機能がpom.xmlファイルに書いてある部品を取ってくる。
欲しいものリスト。
「Spring」を利用するために必要なファイル。


・DemoAppricationファイル
アプリを起動するのに使うファイル

・application.propertiesファイル
RDSとこのプロジェクトを接続するための設定を記載したファイル。

・User.javaファイル
「Javaコード」でSQLのテーブルとテーブルの中身を定義することができるファイル
