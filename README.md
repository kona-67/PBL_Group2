# PBL_Group2


・srcディレクトリ
バックエンド：Javaファイルを入れる場所。


・targetディレクトリ
コンパイルファイルをディレクトリ別に格納している。
コンパイルファイル：Javaファイルをコンピューターが理解できる文章に書き直したファイル。
コンピューターが使うファイルが入っているので、僕らは使いません。


・pom.xmlファイル
Mavenという機能がpom.xmlファイルに書いてある部品を取ってくる。
欲しいものリスト。
「Spring」を利用するために必要なファイル。


・DemoAppricationファイル
アプリを起動するのに使うファイル


・User.javaファイル
「Javaコード」でSQLのテーブルとテーブルの中身を定義することができるファイル
